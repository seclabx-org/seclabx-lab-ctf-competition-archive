# SeclabX CTF 复现归档

本仓库用于 CTF 公开题目的复现与归档，服务于教学与学生练习。
所有内容仅供学习与研究使用。

## 协作入口

- 贡献指南：CONTRIBUTING.md
- 赛事收录任务模板：.github/ISSUE_TEMPLATE/competition.md
- PR 模板：.github/PULL_REQUEST_TEMPLATE.md

## 题目命名规范

```
目录结构：赛事名/年份/题目类型-题目名
赛事名：保留官方大小写，空格用短横线
年份：4 位
题目类型：web | pwn | reverse | crypto | misc | forensics
题目名：小写 + 短横线 + 无空格/中文
示例：
LilCTF/2025/web-ez-sql
GeekGame/2024/reverse-baby-crack
XCTF/2023/pwn-heap-baby
```
